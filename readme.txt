Website thương mại bán sách BookShop nhìn chung đáp ứng được nhu cầu mua sách cho khách hàng với các tính năng:
- Đăng nhập, đăng kí tài khoản.
- Đăng nhập bằng tài khoản facebook.
- Đăng xuất.
- Khách hàng có thể tuỳ chọn mua hàng mà ko cần đăng nhập.
- Có thể cập nhật tài khoàn khách hàng.
- Có thể xem danh sách sản phẩm theo bộ lọc(top hot, top sale, theo nhà xuất bản hay tác giả,...).
- Có thể liên hệ với Admin trên trang liên hệ.
- Sau khi đặt hàng, sẽ có mail báo tới khách hàng.
- Có chức năng tạo tin tức
---------------------------------------------------------------------------------------------------------------
Các bước chuẩn bị để deploy project
B1: trong file có sẵn database, tên: Web_OnlineShop.bak
    Các bạn vào Sql Server rồi restore database là ok. 
B2: Các bạn vào file Web.Config trong Project BookShop cấu hình lại ConnectionString
B3: Các bạn vào file Web.Config trong Project Model cấu hình lại ConnectionString giống như bước 2.
Vậy là ok. Project đã chạy được.
B2 và B3 bạn nào ko biết làm tham khảo web sau để biết thêm chi tiết nhen: 
https://dzone.com/articles/code-first-approach-with-aspnet-mvc-framework
